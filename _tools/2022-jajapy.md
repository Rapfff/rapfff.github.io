Short Description
-----------------
*jajapy* is a python library implementing the Baum-Welch algorithm on various kinds of Markov models.

Selected Features
-----------------
* Learning **HMMs**, **MCs**, **MDPs** and **CTMCs** from traces.
* Parameter estimation for synchronous composition of CTMCs.
* Parameter estimation for **PCTMCs**.
* Compatibility with [Prism](http://www.prismmodelchecker.org/) and [Storm](https://www.stormchecker.org/).
